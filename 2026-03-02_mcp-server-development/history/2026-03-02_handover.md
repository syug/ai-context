## 重要な判断ログ

### Lambda直接呼び出しアーキテクチャ
- REST APIではなくLambda直接呼び出し。IAM SigV4署名で認証。

### IIBS認証パス
- midwayFetch → IIBS → IAM一時クレデンシャル → Lambda invoke

### AISmith IAM Role の共用
- `arn:aws:iam::132046614119:role/AISmith-user-role-prod` を使用

### エージェント委任の失敗
- bypassPermissionsモードのエージェントが3回Write/Bash権限で停止。メインスレッドで直接実装。
